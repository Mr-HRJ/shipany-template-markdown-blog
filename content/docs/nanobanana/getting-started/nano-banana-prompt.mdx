# 主流提示词库与资源站

 综合型 Nano Banana 提示词库，提供大量分类示例、社区共享资源，可按主题查找并复制使用。
🔗 **[nanobananaprompt.org](https://nanobananaprompt.org/)**

## 2. Nano Banana 提示词库

🔗 **[nanoprompts.org](https://nanoprompts.org/zh/)**
