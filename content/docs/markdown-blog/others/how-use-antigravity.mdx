### 一、准备一个“非中国区”的 Google 账号



Gemini 和 Antigravity 并未向中国大陆开放，所以你首先需要一个地区不在中国的 Google 账号。



你有两种方式选择：



#### 方式一：将已有 Google 账号改为其他地区



1. 打开 Google 账号归属地查询网站：  

   [https://policies.google.com/country-association-form](https://policies.google.com/country-association-form)



<img src="https://pub-8dc0158e77a140d4b502b52ab75765b5.r2.dev/docs/0feed9183512999d.jpg" alt="image" />





2. 操作注意点：

   - 开启科学上网，并切换到目标地区（推荐美国，因为有些地区无法申请学生优惠，如中国香港、新加坡不行）。

   - 理由选择最后一个：“其他”。

   - 说明示例：“我想使用 Antigravity，但该应用未在我当前地区开放，因此申请更改地区。”

3. 提交后等待审核即可。



#### 方式二：重新注册一个 Google 账号（更推荐）



1. 注册前先把 VPN 切换到目标地区，**强烈推荐选择美国**。

2. 注册要点：

   - 生日必须设置为 **18 岁以上**（未成年会影响 Gemini 和 Antigravity 使用）。

   - 可以使用中国手机号注册，不影响地区判断。

   - 在手机上注册可直接接收 Google 验证短信，成功率更高（电脑注册可能需扫码并发送海外短信，容易失败）。



<img src="https://pub-8dc0158e77a140d4b502b52ab75765b5.r2.dev/docs/67ecd40d7e2c87a2.jpg" alt="image" />





**如果你赶时间，推荐直接注册新账号，更省心。**

### 三、绑定支付方式（非常关键）



Google 在订阅时会要求绑定支付方式。分三种情况处理：



#### 情况一：你有中国发行的 Visa 信用卡



很多人遇到提示“卡不可用”。



✅ 正确做法：



1. 打开「Google 账号管理」

2. 进入「付款与订阅」

3. 在「付款方式」中先添加 Visa 卡

4. 再回到订阅页面刷新即可正常使用



<img src="https://pub-8dc0158e77a140d4b502b52ab75765b5.r2.dev/docs/cced0cf663c85a27.jpg" alt="image" />





🔗 若仍失败，尝试在 Google Wallet 中绑定：  

[https://wallet.google.com/wallet/paymentmethods](https://wallet.google.com/wallet/paymentmethods)



<img src="https://pub-8dc0158e77a140d4b502b52ab75765b5.r2.dev/docs/9a7b75b43f55f08d.jpg" alt="image" />





#### 情况二：你没有信用卡



可选择：



- 在闲鱼购买一张 VCC 虚拟信用卡（20–30 元）

- 卡内通常自带 1 美元，用于验证即可

- ❌ 千万不要往里面充值（虚拟卡非本人信息，存在风险）



#### 情况三：使用 Google 礼品卡



- 适合长期付费用户

- ❌ 不推荐用于“免费试用/学生优惠”，因为充值金额不能退回

#### 常见问题：登录 Google 账号之后没反应



很多用户会遇到：浏览器完成 Google 登录，返回 Antigravity 时无反应、无提示。



<img src="https://pub-8dc0158e77a140d4b502b52ab75765b5.r2.dev/docs/afd93748b72ab675.jpg" alt="image" />





🔍 原因分析：



1. Antigravity 对可使用地区有限制（中国大陆和香港无法使用）

2. Google 账号未满 18 岁

3. Google 账号归属地不对



#### 解决方案：



- **账号年龄问题**：重新注册最快（修改年龄常不成功）

- **归属地问题**：参考第一章修改账号归属地

- **地区限制问题**：开启科学上网软件的 **TUN 模式**



💡 什么是 TUN 模式？



TUN 模式是“把你手机上所有 App 的网络流量都通过你的梯子来走”的一种工作模式。



一般代理工具默认只代理浏览器等部分 App，而像 Antigravity 这类应用可能会绕过代理。



TUN 模式的作用就是：**强制让 Antigravity 必须走代理线路**，从而绕过地区限制，正常登录使用。



🔧 推荐工具：



若你的软件没有 TUN 模式，可下载 **Clash Verge**：



👉 下载地址：[https://github.com/clash-verge-rev/clash-verge-rev/releases](https://github.com/clash-verge-rev/clash-verge-rev/releases)



配置好后，开启“虚拟网页模式”（即 TUN 模式），然后重新登录即可成功进入。



<img src="https://pub-8dc0158e77a140d4b502b52ab75765b5.r2.dev/docs/167dcb3cdcb7450c.jpg" alt="image" />

## Antigravity Tools



[https://github.com/lbjlaq/Antigravity-Manager](https://github.com/lbjlaq/Antigravity-Manager)



这是一个把 Antigravity 里面的模型转成标准 API，给 Claude Code 等 Coding Agent 接入的智能代理项目。提供多账号管理、协议转换和智能请求调度等功能，让你能稳定、低成本地在 Claude Code、Codex 中使用 gemini / claude 系列模型。



如何使用？



1. 访问 Antigravity Tools 代码仓库，按照指示安装 Antigravity Tools 桌面软件

2. 在 Antigravity Tools 桌面软件添加账号，打开浏览器通过谷歌账号登录 Antigravity

3. 在终端配置环境变量，让 Claude Code 使用自定义的 API 端点



```bash
export ANTHROPIC_API_KEY="sk-xxx"

export ANTHROPIC_BASE_URL="http://127.0.0.1:8045"

```


4. 打开 Claude Code 发送指令，开始使用 CC



有哪些限制？



在 Antigravity Tools  里可以添加多个 Google 账号，每个账号都有一定的 Antigravity 模型额度，如果额度不够了，可以点击切换账号，智能切换到额度足够的账号。



可以为你添加的账号升级 Antigravity 高级套餐，获得更高的额度，既能在 Antigravity 编辑器使用，也能在 Claude Code、Codex 使用，相当于一次充值，同时分配给多个编程智能体用。



可以用哪些模型？



Antigravity 免费账号主要支持的是 gemini / claude 系列模型，不支持 gpt 模型，如果在 Codex 接入，需要加一个模型映射，比如把 gpt-5-codex 映射到 gemini-3-pro-high



这个项目目前只提供桌面版软件，不支持 Web 应用，不能通过服务器部署做 API 中转站，仅供自己在本地使用，一定程度降低了对 Claude、ChatGPT 的账号依赖，仅需一个 Google 账号，即可使用 Antigravity、Claude Code、Codex、Gemini Cli 等编程智能体。
