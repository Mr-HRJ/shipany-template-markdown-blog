# 2025-12-18 （v1.6.3）



feat: 新增注册用户来源统计和数据库 Schema 切换功能

feat: 升级依赖项并添加新的“更新”部分



> 注意：之前已经写业务的项目想更新的话，需要在同步完代码后，执行下面的手动执行sql，因为你的数据库已经执行过`pnpm db:migrate`， drizzle 会跳过它（除非你执行`pnpm db:push`强推）。

> 在 Supabase 控制台的 SQL Editor 中执行  `下面sql` 的内容即可（新增了 utm_source 和 ip、locale 字段)）（如下图）。



```

ALTER TABLE "user" ADD COLUMN IF NOT EXISTS "utm_source" text DEFAULT '' NOT NULL;

ALTER TABLE "user" ADD COLUMN IF NOT EXISTS "ip" text DEFAULT '' NOT NULL;

ALTER TABLE "user" ADD COLUMN IF NOT EXISTS "locale" text DEFAULT '' NOT NULL;

```
