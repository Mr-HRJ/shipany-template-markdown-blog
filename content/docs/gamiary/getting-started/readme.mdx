该模版是一款游戏模板，复刻自`https://gamiary.com/`，支持后台录入管理游戏分类和游戏；

https://gamiary-cf.16781678.xyz/ [自部署到Cloudflare]

## 分支



- `main`:  main branch(部署在Vercel上的分支)

- `cloudflare`:  cf branch (部署在Cloudflare上的分支)



##  注意



> 1、模板中Logo我没有替换成竞品的，避免纠纷，你可以使用Gemini NanoBanana设计个符合游戏主题的logo

> 2、有些游戏即便是在复刻网站上也是不能直接玩的，例如：`https://gamiary.com/game/holiday-mahjong-dimensions/`

> 下图翻译为：【Arkadium 的 Holiday Mahjong Dimensions 游戏目前无法在此处游玩。如果您想玩 Arkadium 的 Holiday Mahjong Dimensions，请点击此处开始游戏。】

> <img src="https://pub-8dc0158e77a140d4b502b52ab75765b5.r2.dev/docs/e1cff6f467f739bf.jpg" alt="image" />

> 遇到这种游戏，建议你删除掉或者给他找一个能玩的平替url编辑替换掉
