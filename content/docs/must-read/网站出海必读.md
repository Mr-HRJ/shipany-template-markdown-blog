# 出海工具站 / 小 SaaS 实战指南（README）

这是一个围绕 **「找需求 → 做产品 → 上线 → 收款 → SEO & 增长 → 复盘」**的完整实战资料合集，目标是帮助你**从 0 到 1 跑通出海网站/工具站的完整闭环**，少踩坑、少试错。


### 1、需求与产品

* 多渠道找需求：词根、榜单、竞品、插件抱怨、趋势、新词监测
* 需求验证与定价：卖空气、定价策略、用户价值判断
* 产品设计：核心链路、转化、低竞争词打法

### 2、数据分析

* Google Analytics / GSC 实战
* 事件上报、用户分群、热力图
* Stripe 数据分析 & 反向用数据驱动内容和页面

### 3、开发与部署

* 模板化上站 SOP，一键部署
* Next.js / 数据库 / 存储 / 性能与安全
* 多语言、登录、客服、反薅羊毛
* AI 辅助开发（用嘴编程、自动化命令）

### 4、支付与收款

* Stripe 从注册到提现全流程
* 无需登录支付、订阅、续费、风控
* 香港银行卡 & 手续费优化

### 5、SEO 与内容

* 关键词研究、页面结构、索引加速
* 图片 SEO、AI 搜索、内容生成与重写
* 对标分析、站内检查、技术 SEO

### 6、外链与增长

* 高质量外链获取方法
* Chrome 插件、聚合站、一次多外链
* SEO + 外链协同打法

### 7、流量与营销

* Twitter / Reddit / YouTube / 邮件营销
* Build in Public、联盟推广、裂变
* 截流、蹭词、活动营销实战

### 8、Ads 投流

* Google Ads 从 0 到 1
* 关键词选择、竞品广告分析
* 监控、目标设置、机器学习优化

### 9、工具与杂项

* 常用神器、插件、性能与实现分析
* 出海工具汇总、变现方式补充

### 10、复盘与认知

* 真实出海项目复盘
* 从 0 赚美元的完整经验
* 技术 × 认知 × 长期主义的思考

## 适合谁？

* 独立开发者 / 工具站站长
* 想做出海网站、AI 工具、小 SaaS 的人
* 不想只学“点技巧”，而是想跑通**完整闭环**的人

## 核心理念

> **需求优先，验证先行，小步快跑，数据驱动，长期复利。**
