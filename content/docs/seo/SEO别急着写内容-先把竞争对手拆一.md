很多人做SEO时，第一反应都是先写内容。

但我越来越觉得，真正靠谱的顺序应该反过来：**先把竞争对手研究透，再决定写什么、做什么、优先优化什么。**

原因很简单。SEO不是闭门造车。你要抢的是搜索结果里的位置，而这些位置现在已经被别人占着了。你如果连对手是谁、他们怎么拿流量、他们强在哪、弱在哪都没搞清楚，就急着写内容，大概率会走很多弯路。

这篇文章要解决的，就是这个问题。

# 一、什么是SEO竞争对手分析

SEO竞争对手分析，本质上就是对竞争对手的SEO策略做系统研究。

目的很明确：

* 找出他们做对了什么
* 找出他们没做好什么
* 判断哪些策略值得借鉴
* 判断哪些位置有机会超过去

说白了，**SEO竞争对手分析不是为了“看看别人”，而是为了指导我接下来该怎么赢。**

---

# 二、为什么要进行SEO竞争对手分析

做这件事，价值非常直接。

通过SEO竞争对手分析，我可以更快搞清楚下面这些问题：

* 这个行业里，哪些SEO打法有效，哪些无效
* 竞争对手有哪些短板可以利用
* 竞争对手有哪些优势值得复制
* 我的SEO任务该怎么排优先级
* 我要超过他们，大概要付出多大成本和难度

很多时候，最省时间的办法，不是自己从零试错，而是先看别人已经试出了什么结果。

---

# 三、什么时候该做SEO竞争对手分析

这件事不是只做一次就完了，但以下几个时间点尤其值得做。

## 1. 你刚起一个新网站

新站最怕瞎做。这个阶段如果先把竞争对手研究清楚，能少走很多没必要的路。

## 2. 你在规划SEO策略

当你准备做内容规划、页面结构设计、关键词布局时，竞争对手分析应该先做。

## 3. 竞争对手超过了你，或者你的关键词排名在掉

这种时候更应该回头看对手。因为排名波动往往不是凭空发生的，很多线索都能从竞争对手的动作里找到。

# 四、如何进行SEO竞争对手分析

为了把流程讲清楚，这里用一个具体案例来演示。

假设我现在要做的是一个新的**信息图表设计工具网站**，下面就是我会走的一整套SEO竞争对手分析流程。

---

## 1. 识别SEO竞争对手

先明确一个概念：**SEO竞争对手，不一定等于商业竞争对手。**

SEO竞争对手，指的是那些在自然搜索里和我争同一批目标关键词的网站。

比如，HubSpot可能在“how to make infographics”这个词上排得很高，但它并不是信息图表设计工具网站的直接商业竞品。可在SEO层面，它就是竞争对手。

### 方法一：如果我已经有网站

操作路径：

1. 打开Ahrefs Site Explorer
2. 输入我的域名
3. 进入Organic competitors报告

这个报告会列出和我有相同关键词排名前10的网站。通常这里面，很多就是我真正该盯的SEO竞争对手。

### 示例数据

目标域名：**venngage.com**

| Domain         | Keyword overlap | Competitor's keywords | Common keywords | Share | Target's keywords |
| -------------- | --------------- | --------------------- | --------------- | ----- | ----------------- |
| visme.co       | -               | 289,393               | 113,399         | 20.4% | 151,827           |
| piktochart.com | -               | 41,185                | 38,580          | 12.6% | 226,646           |
| pitch.com      | -               | 17,034                | 14,586          | 5.2%  | 250,640           |

从这个表里就能快速看出，哪些网站和目标站点在关键词层面重合最多。

### 方法二：如果我是新网站，还没有域名数据

操作路径：

1. 打开Ahrefs Keywords Explorer
2. 输入潜在客户可能会搜索的核心词
3. 查看Traffic share by domain报告

### 示例数据

关键词：**infographic maker, infographic creator**

| Traffic | Share | Domain             | Keywords                               |
| ------- | ----- | ------------------ | -------------------------------------- |
| 2,958   | 25%   | *.canva.com/*      | infographic maker, infographic creator |

| 918     | 8%    | *.adobe.com/*      | infographic maker, infographic creator |

这个报告的意义很大，因为它能直接告诉我：**这些关键词现在的流量，主要被哪些域名吃走了。**

### 筛选真正有效的竞争对手

这一步很重要。不是看到谁排名高，就把谁都当作主攻对象。

像Adobe、Canva这种品牌，本身DR极高，品牌权威、链接资产、内容积累都不是普通网站短期能硬碰硬打过去的。所以这类网站要看，但不一定适合当主要对标对象。

可以用Ahrefs Batch Analysis查看它们的DR。

* Visme

* Adobe

**竞争对手分析的第一步，不是找最强的，而是找最值得参考、也最有机会打赢的。**

## 3. 查找并填补内容空白

接下来，我要找的是**内容差距**。

所谓Content Gap，就是竞争对手已经有排名，而我还没有排名的关键词。

这一步特别重要，因为它可以直接告诉我：**我现在缺哪些页面、缺哪些主题、缺哪些内容方向。**

### 操作方法

1. 打开Ahrefs Competitive Analysis
2. 在Target里输入我的域名
3. 在Competitors里输入竞争对手域名
4. 点击Compare
5. 进入Content Gap报告
6. 勾选Main positions only，排除SERP特性干扰

### 示例数据

目标域名：**newinfographicsmaker.com**
竞争对手：**venngage.com、piktochart.com、visme.co**

| -------------------- | -- | ------ | -- | ---- | ------------------------ | ----------------------- | -------------------------- | ------------------------- | -------------------- | ------------------- |
| booklet              | 3  | 414.0K | 42 | 1.65 | -                        | -                       | -                          | -                         | 10                   | 1                   |
| + ai image generator | 2  | 277.0K | 77 | 0.45 | -                        | -                       | -                          | -                         | 9                    | 996                 |
| + symbols            | 1  | 121.0K | 83 | 0.18 | -                        | -                       | -                          | -                         | 7                    | 2,682               |
| + venn diagram       | 3  | 116.0K | 42 | 0.69 | -                        | -                       | -                          | -                         | 6                    | 2,316               |
| +business proposal   | 3  | 102.0K | 64 | 1.04 | 9                        | 2,484                   | 76                         | 0                         | 10                   | 1,744               |
| + infographic        | 3  | 100.0K | 91 | 1.84 | 691                      | 19,173                  | 7                          | 1,446                     | 2                    | 12,462              |

这类表的核心，不是看到词就做，而是继续筛。

### 如何筛相关关键词

我通常会重点看三件事：

* 这个词和产品是否强相关
* 搜索意图是不是我能接住
* 这个词的难度和商业价值是否值得投入

### 优质目标关键词示例

* 关键词：**infographic examples**
* 搜索量：**20.0K**
* CPC：**0.65**

这类词就很有代表性。它既和产品相关，又有明确内容承接空间。

### 更精准的筛选方法

在Competitors’ positions下拉菜单里，选择**All in top 10**，就能筛出所有竞争对手都排进前10的关键词。

这类词通常更值得优先看。因为一旦多个竞争对手都在吃，往往说明：

* 这个词有稳定需求
* 这个词和行业核心业务强相关
* 这个词是被验证过的SEO机会

## 4. 查看竞争对手的精选摘要

Featured snippets，也就是精选摘要，是Google从排名前10的页面里抽出来的快速答案展示。

如果我自己的页面已经在前10，那就有机会通过重新优化页面结构，把这个位置抢下来。

### 操作方法

1. 打开Ahrefs Competitive Analysis
2. 在Target里输入竞争对手域名
3. 在Competitors里输入我的域名
4. 点击Compare
5. 进入Content Gap报告
6. 在SERP features里选择Where target ranks
7. 勾选Featured snippet
8. 把Target’s position设为Any

### 示例数据

竞争对手：**venngage.com、piktochart.com**

| Keyword      | SF | Volume | KD | CPC  | venngage.com/ - Position | venngage.com/ - Traffic | piktochart.com/ - Position | piktochart.com/ - Traffic |
| ------------ | -- | ------ | -- | ---- | ------------------------ | ----------------------- | -------------------------- | ------------------------- |
| infographic  | 3  | 100.0K | 91 | 1.84 | 1                        | 19,173                  | 7                          | 1,446                     |
| infograph    | 3  | 5.1K   | 92 | 1.56 | -                        | 1,100                   | 7                          | 3                         |
| info graphic | 3  | 4.4K   | 93 | 1.56 | 691                      | 547                     | 8                          | 48                        |

### 页面优化示例

Venngage拿到了“how to make posters”的精选摘要，Google抓取的是一个步骤列表：

1. Identify the goal of your poster....
2. Consider your target audience....
3. Decide where you want to share your poster...
4. Start with a professional poster template....
5. Pick a relevant or branded color scheme....
6. Include a clear call-to-action....
7. Use fonts to create a hierarchy of information.

这就说明，Google很偏好这种**清晰、可提取、结构化的步骤答案**。

如果我的页面目标也是这个关键词，那我就不该只写一段大白话，而应该在正文里加上明确步骤结构，比如：

* 用H3做步骤标题
* 直接编号
* 每步一句先说结论
* 再补充解释

**想抢精选摘要，不是内容更多就行，而是结构要更适合Google抽取。**

## 5. 查看竞争对手的流量来源

SEO分析不能只看关键词，还要看地理分布。

因为有时候，一个站的自然流量很大，不一定全来自英语市场。你如果知道流量主要来自哪些国家，就能判断：

* 哪些语言值得做
* 哪些地区值得单独建页
* 是否要做多语言站点

### 操作方法

1. 打开Ahrefs Site Explorer
2. 输入竞争对手域名
3. 查看Traffic by country

### 示例数据

目标域名：**venngage.com**

| Country       | Traffic | Share | Keywords |
| ------------- | ------- | ----- | -------- |
| United States | 447.8K  | 17.2% | 259.9K   |
| Mexico        | 328.1K  | 12.6% | 60.8K    |
| Philippines   | 273.8K  | 10.5% | 77.1K    |
| India         | 249.3K  | 9.6%  | 120.4K   |
| Brazil        | 239.2K  | 9.2%  | 47.1K    |

这个表说明什么？

说明这个站虽然是英文产品，但它的自然流量并不只来自美国，而是有很大一部分分布在：

* 墨西哥
* 菲律宾
* 印度
* 巴西

### 优化思路

如果我是这个品类的新站，我会得出两个优先级判断：

#### 第一优先级

继续做英语内容，因为美国仍然是占比最高的核心市场。

#### 第二优先级

考虑把核心页面和落地页做多语言扩展，比如：

* 西班牙语，对应墨西哥
* 葡萄牙语，对应巴西
* 他加禄语，对应菲律宾
* 印地语，对应印度

如果资源够，还可以逐步开多语言博客。

**很多时候，SEO增长不只是多写内容，而是把同样的内容，放到更多能吃到流量的语言市场里。**

## 6. 查找反向链接差距

链接仍然是Google很重要的排名因素之一。

所以我会去看：**竞争对手拿到了哪些域名的外链，而我还没有。**

这一步的价值在于，它能直接给我一份“可争取的外链名单”。

### 操作方法

1. 打开Ahrefs Site Explorer
2. 输入我的域名
3. 进入Link Intersect报告
4. 顶部输入竞争对手域名
5. 底部输入我的域名
6. 点击Show link opportunities

### 示例数据

3个竞争对手 vs **newinfographicsmaker.com**

| Referring domain | DR | Ahrefs rank | Intersect | venngage.com/ | piktochart.com/ | visme.co/ |
| ---------------- | -- | ----------- | --------- | ------------- | --------------- | --------- |
| google.com       | 98 | 6           | 3         | 10            | 8               | 4         |
| interest.com     | 97 | 8           | 3         | 8             | 2               | 7         |
| adobe.com        | 96 | 28          | 3         | 3             | 6               | 2         |
| github.com       | 96 | 21          | 3         | 17            | 12              | 15        |
| microsoft.com    | 96 | 18          | 3         | 3             | 7               | 1         |
| me               | 96 | 29          | 3         | 40            | 28              | 8         |
| reddit.com       | 95 | 46          | 3         | 3             | 2               | 8         |

这份表里最重要的不是盯着大站流口水，而是去找**可以复制的具体链接机会。**

### 可复制的外链机会示例

比如点击**cnet.com**这个来源，会发现竞争对手出现在它的“可测试工具列表”里。

对应链接示例：

* Venngage链接：download.cnet.com/enngage/3000-0247_4-77503278.html
* Visme链接：download.cnet.com/Visme/3000-2075_4-76145257.html

这类机会就非常具体。因为它不是抽象地说“去做外链”，而是告诉我：

* 这个站接受同类工具收录
* 竞争对手已经上去了
* 我也有机会申请进去

这才是能执行的外链分析。

## 7. 发现链接诱饵机会

所谓链接诱饵，说白了就是那种**特别容易吸引别人主动引用和链接的内容**。

这类页面本身不一定直接转化，但它们能拿很多外链，然后再通过内部链接把权重传给真正的转化页。

### 操作方法

1. 打开Ahrefs Site Explorer
2. 输入竞争对手域名
3. 进入Best by links报告

### 示例数据

目标域名：**venngage.com**

| Target page                                                                                                                                                                                      | UR | Referring domains | Top DR domains | Links to page |
| ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ | -- | ----------------- | -------------- | ------------- |
| Venngage | Professional Infographic Maker | 10,000+ Templates（[https://venngage.com/）](https://venngage.com/）)                                                                                   | 42 | 8,392             | 98             | 77,096        |
| 16 Visual Content Marketing Statistics for 2022 [Infographic]（[https://venngage.com/blog/visual-content-marketing-statistics/）](https://venngage.com/blog/visual-content-marketing-statistics/）) | 33 | 1,998             | 99             | 3,939         |

从这里可以看出来，除了首页这种天然会拿链接的页面之外，**统计类内容**也很能吸链接。

比如：

* 行业统计
* 数据报告
* 信息图汇总
* 趋势盘点

结论很明确：**统计类内容和可视化内容，是这个赛道里很典型的高效链接诱饵。**

如果我要做类似站点，这类内容值得专门规划。

## 8. 查找竞争对手的失效网页

这一招很实用，也经常被低估。

逻辑是这样的：竞争对手有些页面已经404了，但这些页面过去拿到的外链还在。那我就可以：

1. 看看原页面以前讲了什么
2. 自己做一个更好的替代页面
3. 去联系那些还在链接404页的网站
4. 请他们把链接换到我的新页面上

### 操作方法

1. 打开Ahrefs Site Explorer
2. 输入竞争对手域名
3. 进入Best by links报告
4. 把HTTP code筛选为404 not found

### 示例数据

目标域名：**visme.co**

| Target page                                                                                                                                                                                                | UR | Referring domains | Top DR | Links to page |
| ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | -- | ----------------- | ------ | ------------- |
| [https://visme.co/blog/most-common-birthday/](https://visme.co/blog/most-common-birthday/)                                                                                                                 | 12 | 60                | 92     | 161           |
| [https://visme.co/blog/gestalt-design-principles/](https://visme.co/blog/gestalt-design-principles/)                                                                                                       | 9  | 33                | 94     | 38            |
| [https://visme.co/blog/amazing-leaders-who-once-had-crippling-stage-fright-and-how-they-overcame-it/](https://visme.co/blog/amazing-leaders-who-once-had-crippling-stage-fright-and-how-they-overcame-it/) | 6  | 25                | 94     | 42            |

### 实际操作步骤

#### 第一步

点击目标页面的**View on Archive.org**，去Wayback Machine看历史页面内容。

#### 第二步

做一个同主题、但质量更高的替代页面。

#### 第三步

联系那些还在给404页做链接的网站，请他们把链接改到我的页面。

这一招的好处是，外链目标已经现成了，沟通理由也很自然，因为你是在帮对方修失效链接。

## 9. 查看竞争对手的Core Web Vitals

技术层面也要看。因为SEO不只是内容和外链，页面体验同样会影响表现。

Core Web Vitals是Google页面体验信号的一部分，所以我会把自己的网站和竞争对手做对比。

### 方法一：单页面分析

直接把竞争对手页面丢进PageSpeed Insights。

### 示例数据

页面：**[https://venngage.com/](https://venngage.com/)**
统计时间：**Jul 18, 2023**

| 指标                           | 数值     |
| ---------------------------- | ------ |
| Core Web Vitals Assessment   | Failed |
| Largest Contentful Paint LCP | 3.1s   |
| First Input Delay FID        | 19 ms  |
| Cumulative Layout Shift CLS  | 0      |

如果竞争对手页面体验做得一般，而我的页面更快、更稳，那这就是技术上的超车机会。

### 方法二：批量分析

更高效的做法，是用Ahrefs Site Audit抓整个竞争对手站点，并连接PageSpeed Insights API，批量看技术问题。

### 示例数据

桌面端，**4,661个页面**

| 指标                  | 数值              |
| ------------------- | --------------- |
| Core Web Vitals表现良好 | 1,519个页面        |
| 需要改进                | 1,120个页面        |
| 表现较差                | 60个页面           |
| 有CrUX指标的页面          | 131个，减少5个       |
| Lighthouse平均得分      | 63良好、46需改进、22较差 |

如果我看到竞争对手大批页面体验一般，这就意味着：**技术优化不是补作业，而可能是真正的排名突破口。**

## 10. 查看竞争对手在付费搜索中的关键词

很多人做SEO，只盯自然词，不看付费词。但其实竞争对手在投广告的关键词，往往有很高的商业价值。

因为没人会长期给没有价值的词一直烧钱。

## 11. 学习竞争对手的PPC广告文案

除了看他们投哪些词，我还会看他们广告是怎么写的。

因为PPC广告文案本质上就是经过点击率和转化率检验的卖点表达。很多时候，它对SEO标题和描述也有借鉴价值。

### 操作方法

1. 打开Ahrefs Site Explorer
2. 输入竞争对手域名
3. 进入Paid keywords报告
4. 鼠标悬停在关键词旁边的放大镜图标上
5. 查看广告文案

### 示例广告

Venngage针对“infographic design”的广告文案：

> #1 Infographic Maker - Make Better Infographics
> Our customizable templates help you become a design wiz in a few clicks. Sign up for free! Make infographics that people love. Sign up & join 40,000 businesses who use Venngage...

### 我会怎么拆这个广告

#### 核心卖点词

* customizable
* few clicks
* design wiz

#### 行动号召

* Sign up for free

#### 社会证明

* 40,000 businesses

这类信息很有价值，因为它已经告诉我：用户对这类产品更容易被什么打动。

如果我在做SEO标题、元描述、页面首屏文案时，适当吸收这些表达，通常也更容易提升点击率和转化率。

# 最后

SEO竞争对手分析，远不止这11步。

但如果你能先把上面这些动作做完，其实已经足够把很多核心情报摸清楚了。

我自己做这件事时，最强烈的感受是：**竞争对手分析不是SEO的辅助环节，而是SEO策略制定前的必修课。**

因为很多关键答案，都不需要我凭空猜。

我完全可以从竞争对手那里直接看到：

* 什么关键词值得做
* 什么页面结构在吃流量
* 什么内容类型更容易拿链接
* 什么市场值得扩语言
* 什么商业词值得优先布局
* 什么技术问题可能成为超车点

最后真正重要的，不是看完这些分析有多兴奋，而是把分析结果用到执行里。

说到底，SEO竞争对手分析只是起点。**真正拉开差距的，还是后面的执行。**
